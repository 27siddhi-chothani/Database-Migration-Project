# Database Migration Project

## Objective
Migrate data from MySQL to PostgreSQL and ensure data integrity.

## Tools Used
- MySQL
- PostgreSQL
- SQL

## Dataset
- Customers Table
- Orders Table

## Migration Process
1. Created source database in MySQL.
2. Created target database in PostgreSQL.
3. Exported data from MySQL.
4. Imported data into PostgreSQL.
5. Verified records after migration.

## Files Included

- mysql_database.sql
- postgresql_database.sql
- migration_script.sql
- Migration_Report.pdf
- Screenshots

## Outcome

Successfully migrated all records while preserving data consistency.
